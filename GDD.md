# GDD Index

The Ricochet Tanks GDD was moved into structured docs:

- `docs/GDD_RU.md` - main Russian GDD and source of truth.
- `docs/GDD_EN.md` - English translation/adaptation.
- `docs/TECH_STATUS.md` - current implementation status.
- `docs/ROADMAP.md` - next tasks.
- `docs/MOBILE_CONTROLS.md` - mobile landscape controls design.
- `docs/GD_QUESTIONS.md` - guiding questions for game design decisions.

`docs/GDD.md` is kept as a compatibility pointer for older links.
